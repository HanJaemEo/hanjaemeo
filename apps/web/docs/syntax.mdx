## 韓国語の品詞の分類

| 品詞   | コード上での表現 | 色       | 例                                                 | コード                                                     |
| ------ | ---------------- | -------- | -------------------------------------------------- | ---------------------------------------------------------- |
| 名詞   | `noun`           | `pink`   | <Noun>{유행}^(流行)</Noun>                         | `<Noun>{유행}^(流行)</Noun>{:mdx}`                         |
| 動詞   | `verb`           | `green`  | <Verb>{말해줘}^(言って)</Verb>                     | `<Verb>{말해줘}^(言って)</Verb>{:mdx}`                     |
| 形容詞 | `adjective`      | `purple` | <Adjective>{착한}^(善良な)</Adjective>             | `<Adjective>{착한}^(善良な)</Adjective>{:mdx}`             |
| 助詞   | `particle`       | `slate`  | <Particle>{은}^(は)</Particle>                     | `<Particle>{은}^(は)</Particle>{:mdx}`                     |
| 副詞   | `adverb`         | `indigo` | <Adverb>{더}^(もっと)</Adverb>                     | `<Adverb>{더}^(もっと)</Adverb>{:mdx}`                     |
| 代名詞 | `pronoun`        | `orange` | <Pronoun>{난}^(私は)</Pronoun>                     | `<Pronoun>{난}^(私は)</Pronoun>{:mdx}`                     |
| 冠形詞 | `determiner`     | `yellow` | <Determiner>{그}^(その)</Determiner>               | `<Determiner>{그}^(その)</Determiner>{:mdx}`               |
| 数詞   | `numeral`        | `cyan`   | <Numeral>{첫째}^(一つ目)</Numeral>                 | `<Numeral>{첫째}^(一つ目)</Numeral>{:mdx}`                 |
| 感嘆詞 | `interjection`   | `lime`   | <Interjection>{여보세요}^(もしもし)</Interjection> | `<Interjection>{여보세요}^(もしもし)</Interjection>{:mdx}` |
| 未分類 | `unclassified`   | `slate`  | <Unclassified>{지}^(でしょ？)</Unclassified>       | `<Unclassified>{지}^(でしょ？)</Unclassified>{:mdx}`       |
| 空白   | `whitespace`     | `white`  | <Space />                                          | `<Space />{:mdx}`                                          |

> [!WARNING]  
> `unclassified`は、慣用表現などで分類が難しい場合にのみ使用する。

この分類表は、韓国語の品詞をコード上での表現と色で区別し、整理するための基本的なガイドラインを提供します。ただし、言語学の観点から、特定の文脈や詳細な研究によってはさらなる調整が必要になる場合があります。

> 韓国の学校文法では名詞・代名詞・数詞・助詞・動詞・形容詞・冠形詞・副詞・感嘆詞という9つの品詞を設けており、さらに文法成分として体言・関係言・用言・修飾言・独立言を設けている。 https://ja.wikipedia.org/wiki/%E6%9C%9D%E9%AE%AE%E8%AA%9E%E3%81%AE%E6%96%87%E6%B3%95

## 元の文章

```
유행이 돌고 돌아도 난 그 틀에 없어 이미
뭐든 될 대로 되라지
```

## 学習用の表示

유행이 돌고 돌아도 난 그 틀에 없어 이미

> 流行が回り回っても、私はもうその枠にいないの

{유행}^(流行){이}^(が) {돌고}^(回り) {돌아도}^(回っても) {난}^(私は) {그}^(その) {틀}^(枠){에}^(に) {없어}^(いない) {이미}^(もう)

```mdx
{유행}^(流行){이}^(が) {돌고}^(回り) {돌아도}^(回っても) {난}^(私は) {그}^(その) {틀}^(枠){에}^(に) {없어}^(いない) {이미}^(もう)
```

뭐든 될 대로 되라지

> 何事も為るように為るって言うでしょ

{뭐}^(何){든}^(にしろ) {될}^(為る) {대}^(まま){로}^(で) {되라}^(為るって言う){지}^(でしょ)

```mdx
{뭐}^(何){든}^(にしろ) {될}^(為る) {대}^(まま){로}^(で) {되라}^(為るって言う){지}^(でしょ)
```

# LLMプロンプト

あなたは日本人向けの韓国語教師です。与えられた韓国語の文章もしくは会話文に以下の形式で対語訳と意訳をつけてください。

## 単語ごとの対訳と品詞の表示

| 品詞   | 例                                                 |
| ------ | -------------------------------------------------- |
| 名詞   | <Noun>{유행}^(流行)</Noun>                         |
| 動詞   | <Verb>{말해줘}^(言って)</Verb>                     |
| 形容詞 | <Adjective>{착한}^(善良な)</Adjective>             |
| 助詞   | <Particle>{은}^(は)</Particle>                     |
| 副詞   | <Adverb>{더}^(もっと)</Adverb>                     |
| 代名詞 | <Pronoun>{난}^(私は)</Pronoun>                     |
| 冠形詞 | <Determiner>{그}^(その)</Determiner>               |
| 数詞   | <Numeral>{첫째}^(一つ目)</Numeral>                 |
| 感嘆詞 | <Interjection>{여보세요}^(もしもし)</Interjection> |
| 未分類 | <Unclassified>{지}^(でしょ？)</Unclassified>       |
| 空白   | <Space />                                          |

ただし、`unclassified`は、慣用表現や古典口語表現などで分類が難しい場合にのみ使用する。

## 単語の分割

空白で区切られていなくても、助詞や動詞の活用語尾などを見て、適切に分割してください。

- 正しい例

  <Noun>{위}^(上)</Noun>
  <Particle>{에}^(に)</Particle>

- 正しくない例
  <Particle>{위에}^(上に)</Particle>
